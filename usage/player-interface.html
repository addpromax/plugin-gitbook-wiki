玩家界面说明

玩家界面说明

本章为玩家详细介绍如何使用 Codex 图鉴系统，包括界面操作、功能说明和实用技巧。

开始使用

打开图鉴界面

使用以下命令打开 Codex 主界面：

/codex

如果您没有权限使用此命令，请联系服务器管理员。

主界面布局

打开图鉴后，您将看到如下布局的界面：

┌─────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────┤
│  📚          ⚔️          🎣          🌾           │

界面元素详解

分类图标

每个分类都有独特的图标和进度显示：

图标 分类名称 说明
📚 物品收集 记录获得的各种物品
⚔️ 怪物猎人 记录击杀的生物类型
🎣 钓鱼图鉴 记录钓到的鱼类（需要CustomFishing）
🌾 作物图鉴 记录种植的作物（需要CustomCrops）
✨ 附魔图鉴 记录发现的附魔
📖 世界历史 记录世界背景故事

进度显示

每个分类下方显示当前进度：

  • 进度条：██░░ 表示已完成的比例
  • 数字进度：(2/10) 表示已解锁2个，总共10个
  • 百分比：有些服务器还会显示完成百分比

颜色含义

  • 🟢 绿色：已完成的分类
  • 🟡 黄色：进行中的分类
  • ⚪ 灰色：未开始的分类
  • 🔴 红色：锁定的分类（需要特定条件解锁）

发现状态说明

每个发现条目有不同的状态：

✅ 已解锁

  • 外观：正常颜色显示
  • 信息：显示完整的名称、描述和解锁时间
  • 操作：可以查看详细信息

❌ 未解锁

  • 外观：灰色或暗淡显示
  • 信息：显示提示信息，告诉您如何解锁
  • 操作：鼠标悬停查看解锁条件

🔒 锁定

  • 外观：特殊锁定图标
  • 信息：显示解锁所需的前置条件
  • 操作：完成前置条件后自动解锁

❓ 神秘

  • 外观：问号或特殊图标
  • 信息：完全隐藏，只显示"???"
  • 操作：探索游戏以发现

特殊功能

搜索功能

某些服务器可能启用了搜索功能：

  1. 点击搜索图标（🔍）
  2. 输入关键词
  3. 查看匹配的发现条目

分类筛选

使用筛选功能快速找到特定类型的发现：

  • 已解锁：只显示已完成的发现
  • 未解锁：只显示还未完成的发现
  • 最近：按解锁时间排序
  • 稀有度：按稀有程度排序

统计信息

在某些界面可以查看详细统计：

实用技巧

📱 快速导航

  1. 快捷键（如果服务器支持）：

    • 按 TAB 键快速打开图鉴
    • 使用 WASD 键导航分类
    • 按 ESC 键返回上级界面

  2. 书签功能：

    • 右键点击分类可添加到收藏
    • 收藏的分类会出现在主界面顶部

  3. 快速跳转：

    • 点击进度条可以直接跳转到未完成的发现
    • 使用页码可以快速跳转页面

🎯 高效收集策略

  1. 优先级规划：

    • 先完成简单的发现（如基础物品收集）
    • 逐步挑战困难的发现（如Boss击杀）
    • 关注有时间限制的发现

  2. 批量收集：

    • 同时携带多种目标物品
    • 在不同生物群系间移动时留意发现机会
    • 利用自动农场提高效率

  3. 社交合作：

    • 与其他玩家分享发现技巧
    • 组队完成困难的发现
    • 交易稀有物品

🔔 通知设置

关注发现通知，及时了解进度：

  1. 解锁通知：

    🎉 新发现解锁！
✨ 附魔图鉴: 锋利
🎁 获得奖励: 50经验

  2. 里程碑通知：

    🏆 成就达成！
📚 物品收集大师
🎁 特殊奖励已发放到背包

  3. 进度提醒：

    📊 进度更新
🌾 作物图鉴: 8/12 (67%)
💡 还差4种作物就能获得大师称号！

移动端使用

触屏操作

如果您使用移动设备连接服务器：

  1. 点击：相当于鼠标左键
  2. 长按：相当于鼠标右键
  3. 双击：快速操作
  4. 滑动：在大界面中导航

界面适配

移动端界面会自动适配：

  • 更大的按钮和图标
  • 简化的信息显示
  • 触屏友好的导航

个性化设置

界面主题

某些服务器允许玩家自定义界面主题：

  1. 颜色主题：

    • 经典蓝色
    • 自然绿色
    • 魔法紫色
    • 火焰红色

  2. 显示偏好：

    • 详细模式：显示完整信息
    • 简洁模式：只显示关键信息
    • 紧凑模式：更多条目，更少空白

  3. 排序方式：

    • 按解锁时间
    • 按字母顺序
    • 按稀有度
    • 按分类

隐私设置

控制您的发现信息的可见性：

  • 公开：其他玩家可以查看您的所有发现
  • 朋友：只有好友可以查看详细进度
  • 私密：只有自己可以查看完整信息

常见操作问题

界面无法打开

可能原因：

  • 权限不足
  • 服务器卡顿
  • 插件冲突

解决方法：

  1. 联系管理员检查权限
  2. 等待服务器负载降低
  3. 重新登录服务器

发现没有更新

可能原因：

  • 缓存延迟
  • 网络延迟
  • 数据同步问题

解决方法：

  1. 重新打开界面
  2. 等待1-2分钟后再查看
  3. 联系管理员刷新数据

进度显示错误

可能原因：

  • 数据不一致
  • 配置更新
  • 插件版本问题

解决方法：

  1. 记录具体的错误信息
  2. 联系服务器管理员
  3. 提供屏幕截图协助诊断
