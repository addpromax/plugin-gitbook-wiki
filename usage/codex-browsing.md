# 图鉴浏览

本章教您如何高效地浏览和管理 Codex 图鉴系统中的发现条目，充分利用各种功能来提升使用体验。

## 浏览策略

### 📚 系统性浏览

#### 按分类浏览
最推荐的浏览方式是按分类逐一查看：

1. **物品收集** - 从基础材料开始
2. **怪物猎人** - 了解战斗相关发现
3. **附魔图鉴** - 掌握附魔知识
4. **钓鱼图鉴** - 探索水域奥秘
5. **作物图鉴** - 学习农业技巧
6. **世界历史** - 了解背景故事

#### 按难度浏览
```
新手级 → 进阶级 → 专家级 → 大师级 → 传奇级
   ↓       ↓        ↓        ↓       ↓
简单物品  稀有材料  Boss击杀  复合条件  终极挑战
```

### 🎯 目标导向浏览

#### 奖励导向
优先完成奖励丰厚的发现：
- 💎 **经验奖励** - 快速升级
- 💰 **物品奖励** - 获得稀有物品
- 🏆 **称号奖励** - 展示成就
- 🎁 **特殊奖励** - 独特福利

#### 进度导向
关注接近完成的分类：
- 查看进度条接近满格的分类
- 优先完成还差1-2个发现的分类
- 获得分类完成奖励

## 常见浏览问题

### ❓ 找不到特定发现

**解决方法**：
1. 使用搜索功能
2. 检查不同的分类
3. 确认是否已解锁
4. 查看是否有前置条件

### 🐌 界面加载缓慢

**优化方法**：
1. 减少同时显示的项目数量
2. 关闭不必要的动画效果
3. 清理界面缓存
4. 检查网络连接

### 📱 移动端显示异常

**解决方案**：
1. 切换到移动适配模式
2. 调整界面缩放比例
3. 使用简洁显示模式
4. 重启游戏客户端
