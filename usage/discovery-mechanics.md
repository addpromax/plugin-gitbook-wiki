# 发现机制详解

本章为玩家详细解释 Codex 图鉴系统中各种发现的触发机制，帮助您更高效地收集和解锁各种发现条目。

## 发现系统概述

### 什么是发现？

发现是 Codex 系统中的核心概念，代表玩家在游戏中完成的各种成就或里程碑。每个发现都有：

- **名称**：发现的标题
- **描述**：详细说明和背景故事
- **触发条件**：解锁这个发现需要满足的条件
- **奖励**：解锁后获得的奖励
- **稀有度**：发现的珍贵程度

### 发现的类型

根据触发方式，发现主要分为以下类型：

| 类型 | 图标 | 说明 | 示例 |
|------|------|------|------|
| 物品获取 | 📦 | 首次获得特定物品时触发 | 第一颗钻石、稀有宝石 |
| 生物击杀 | ⚔️ | 击杀特定生物时触发 | 屠龙勇士、怪物猎手 |
| 区域探索 | 🗺️ | 进入特定区域时触发 | 神秘洞穴、隐藏神庙 |
| 指令执行 | 💻 | 首次执行特定指令时触发 | 传送大师、指令专家 |
| 附魔发现 | ✨ | 接触到新附魔时触发 | 锋利精通、保护大师 |
| 钓鱼成就 | 🎣 | 钓到特定鱼类时触发 | 渔夫传说、稀有鱼类 |
| 农业成就 | 🌾 | 种植作物时触发 | 农业专家、作物大师 |

## 物品获取发现

### 基础物品发现

最常见的发现类型，当您首次获得某种物品时会自动触发。

#### 💎 宝石收集系列
```
第一颗钻石 → 钻石收藏家 → 宝石大师
     ↓             ↓           ↓
  获得1个钻石   获得64个钻石  收集所有宝石
```

**触发方式**：
- 从挖掘中获得
- 从交易中获得  
- 从其他玩家处获得
- 从战利品箱中获得

**实用技巧**：
- 💡 带上时运附魔的工具挖矿
- 💡 在 Y=11 层挖掘钻石效率最高
- 💡 使用夜视药水提高挖掘效率

#### 🛠️ 工具制作系列
```
石器时代 → 铁器时代 → 钻石工具 → 下界合金
    ↓         ↓          ↓         ↓
制作石工具  制作铁工具  制作钻石工具  制作下界工具
```

**解锁顺序**：
1. 木制工具（新手必经）
2. 石制工具（进入石器时代）
3. 铁制工具（文明发展）
4. 钻石工具（高端装备）
5. 下界合金（终极装备）

### 稀有物品发现

某些发现需要特殊的稀有物品：

#### 🌟 特殊材料
- **龙蛋**：击败末影龙后获得
- **信标**：制作需要下界之星
- **附魔金苹果**：极其稀有，需要8个金块
- **鞘翅**：在末地城中找到

#### 🎁 自定义物品
如果服务器安装了自定义物品插件：
- **魔法武器**：特殊制作或任务奖励
- **传奇装备**：Boss掉落或活动获得
- **收藏品**：限时活动物品

## 生物击杀发现

### 基础怪物击杀

#### ⚔️ 怪物猎手系列
```
僵尸杀手 → 骷髅射手 → 爬行者克星 → 怪物专家
    ↓         ↓           ↓           ↓
 击杀僵尸   击杀骷髅    击杀爬行者   击杀所有基础怪物
```

**战斗技巧**：
- 🛡️ 穿戴盔甲提高生存率
- ⚔️ 使用锋利附魔增加伤害
- 🏹 远程攻击骷髅和爬行者
- 🌙 夜晚怪物生成更多

#### 🔥 下界怪物
- **烈焰人**：在下界要塞中找到
- **凋零骷髅**：概率掉落凋零骷髅头颅
- **僵尸猪人**：和平相处或主动攻击
- **恶魂**：使用弓箭或反弹火球

#### 🐉 Boss 挑战
- **凋零Boss**：需要凋零骷髅头颅x3 + 灵魂沙
- **末影龙**：前往末地挑战
- **监守者**：在海底神庙中找到

### 自定义怪物（需要插件支持）

#### 🌟 MythicMobs 怪物
如果服务器使用 MythicMobs：
- **古龙**：传说级Boss
- **精英守卫**：高等级精英怪
- **元素法师**：魔法系Boss

**挑战建议**：
- 🎯 组队挑战困难Boss
- 💊 准备足够的药水
- 🛡️ 使用最好的装备
- 📍 了解Boss技能和弱点

## 区域探索发现

### WorldGuard 区域

#### 🏰 特殊建筑
- **城堡主厅**：进入城堡核心区域
- **神秘地牢**：发现隐藏地下城
- **古代遗迹**：探索废弃建筑
- **秘密花园**：找到隐藏的美丽区域

#### 🌍 世界探索
- **极地探险**：到达冰原生物群系
- **沙漠绿洲**：在沙漠中找到水源
- **丛林深处**：探索茂密的雨林
- **海底世界**：潜入深海区域

**探索技巧**：
- 🗺️ 使用地图记录已探索区域
- 🧭 携带指南针防止迷路
- 🎒 准备充足的食物和装备
- 🔦 带上火把照明

### 坐标触发区域

某些发现需要到达特定坐标：

```
极北之地: X < -1000, Z < -1000
极南之地: X > 1000, Z > 1000
世界中心: X = 0, Z = 0 (±10格范围内)
高空旅行: Y > 200
深渊探索: Y < 5
```

## 指令执行发现

### 基础指令系列

#### 💻 指令专家
- **传送新手**：首次使用 `/spawn`
- **回家达人**：首次使用 `/home`
- **传送门专家**：首次使用 `/warp`

#### 📊 信息查询
- **统计专家**：使用 `/stats` 查看统计
- **排行榜观察者**：使用 `/top` 查看排行
- **帮助求助者**：使用 `/help` 查看帮助

**注意事项**：
- ✅ 只有首次使用才会触发发现
- ✅ 某些指令需要特定权限
- ✅ 错误的指令不会触发发现

## 附魔发现系统

### 原版附魔发现

#### ⚔️ 武器附魔
```
锋利发现 → 击退发现 → 火焰附加 → 抢夺发现
    ↓         ↓          ↓          ↓
获得锋利附魔 获得击退附魔 获得火附魔 获得抢夺附魔
```

#### 🛡️ 防护附魔
- **保护**：基础防护附魔
- **爆炸保护**：对抗爬行者
- **弹射物保护**：对抗弓箭
- **火焰保护**：对抗火焰伤害

#### 🛠️ 工具附魔
- **效率**：提高挖掘速度
- **时运**：增加掉落物
- **精准采集**：完整采集方块
- **耐久**：减少耐久损耗

### 自定义附魔发现

如果服务器安装了 Aiyatsbus 或其他附魔插件：

#### ✨ 特殊附魔
- **心灵手巧**：方块直接进背包
- **自动熔炼**：破坏时自动熔炼
- **经验提升**：获得额外经验
- **灵魂绑定**：死亡不掉落

**获得方式**：
- 📚 附魔台随机获得
- ⚒️ 铁砧组合附魔书
- 🎣 钓鱼获得附魔书
- 💰 商店购买或交易

## 钓鱼图鉴系统

### CustomFishing 集成

如果服务器使用 CustomFishing 插件：

#### 🐟 鱼类发现
```
普通鱼类 → 稀有鱼类 → 传奇鱼类 → 神话鱼类
    ↓         ↓          ↓          ↓
  容易钓到   需要技巧    需要耐心    极其稀有
```

#### 🌊 钓鱼环境
- **淡水区域**：河流、湖泊
- **海水区域**：海洋、深海
- **特殊区域**：岩浆、特殊液体

**钓鱼技巧**：
- 🎣 使用海之眷顾附魔提高效率
- 🌧️ 雨天钓鱼效果更好
- 🕐 不同时间钓到不同鱼类
- 🌍 不同生物群系有不同鱼类

### 钓鱼统计

系统会记录详细的钓鱼数据：
- **鱼类数量**：每种鱼钓到的总数
- **最大尺寸**：钓到的最大鱼类记录
- **发现时间**：首次钓到的时间
- **稀有度**：鱼类的珍贵程度

## 农业图鉴系统

### CustomCrops 集成

#### 🌱 作物发现
```
基础作物 → 高级作物 → 魔法作物 → 传奇作物
    ↓         ↓          ↓          ↓
  小麦玉米   特殊蔬菜    魔法植物    神话作物
```

#### 🌞 季节系统
某些作物只能在特定季节种植：
- **春季作物**：萝卜、土豆、小麦
- **夏季作物**：玉米、西瓜、南瓜  
- **秋季作物**：胡萝卜、甜菜、苹果
- **冬季作物**：特殊冬季植物

**种植技巧**：
- 💧 确保作物有足够水源
- ☀️ 提供适当的光照
- 🌱 使用骨粉加速生长
- 🕐 注意作物的生长时间

### 农业统计

系统记录农业相关数据：
- **种植次数**：每种作物的种植次数
- **收获次数**：成功收获的次数
- **首次种植**：第一次种植的时间
- **产量统计**：总产量和平均产量

## 发现触发条件

### 时间限制

某些发现有特殊的时间要求：

#### 🌙 时间相关发现
- **夜行者**：只在夜晚触发
- **日出见证者**：在日出时触发
- **午夜传说**：在午夜12点触发
- **节日特别**：特定节日才能触发

#### ⏰ 持续时间要求
- **耐力测试**：需要连续在线一定时间
- **马拉松**：需要连续游戏数小时
- **坚持不懈**：需要连续多天登录

### 复合条件

高级发现可能需要满足多个条件：

#### 🎯 多重条件示例
**传奇冒险家**需要：
1. ⚔️ 击败末影龙
2. 💎 拥有全套钻石装备
3. 🏰 探索10个特殊区域
4. 📊 达到50级经验
5. 🕐 游戏时间超过100小时

### 前置发现

某些发现需要先完成其他发现：

```
新手村民 → 村庄探索者 → 贸易专家 → 经济大师
    ↓           ↓           ↓          ↓
 进入村庄   与村民交易   完成50笔交易 成为富豪
```
